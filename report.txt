* ABSTRACT

Financial research often requires collecting and analyzing company information from various sources. Manual research can be time-consuming and inefficient. The Autonomous Financial Research Agent is a Python-based application that automates the process of retrieving company information and generating basic financial research reports.
The system uses the Yahoo Finance Search API to gather company details such as company name, stock symbol, and exchange information. It then generates a simple research report automatically and stores it in a text file. This project demonstrates the use of APIs, automation, and report generation using Python.

* INTRODUCTION

Financial analysts rely on accurate company information before making investment decisions. Accessing such information manually can be repetitive and time-consuming. Automation can significantly improve efficiency.
The Autonomous Financial Research Agent allows users to enter a company name and automatically retrieves company details from Yahoo Finance. The application then creates a research report containing the gathered information.

Company Information Retrieval Function

def get_company_info(company):
    url = f"https://query1.finance.yahoo.com/v1/finance/search?q={company}"

    try:
        response = requests.get(url)
        data = response.json()

        if "quotes" in data and len(data["quotes"]) > 0:
            result = data["quotes"][0]

            print("\n--- Company Information ---")
            print("Name:", result.get("shortname", "N/A"))
            print("Symbol:", result.get("symbol", "N/A"))
            print("Exchange:", result.get("exchange", "N/A"))
        else:
            print("No company information found.")

    except Exception as e:
        print("Error:", e)
